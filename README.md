Music Web App — це вебзастосунок з інтеграцією Spotify API для прослуховування музики та керування персональною музичною активністю користувача.

У застосунку можна шукати та відтворювати треки, додавати їх до збережених і надалі керувати ними: відтворювати, видаляти або додавати в чергу відтворення. Черга підтримує різні режими роботи — від старіших доданих треків до новіших або навпаки, і систему пріоритетів.

Система пріоритетів базується на частоті прослуховувань треків. Чим частіше користувач слухає певну композицію, тим вищий її пріоритет у черзі. Розрахунок враховує як загальну історію прослуховувань, так і активність за поточний день.

Окремо реалізовано розділ історії прослуховувань, де користувач може переглядати треки за обраною датою. Для кожного треку доступне повторне відтворення, а також можливість створення Spotify playlist на основі вибраного списку. Створеним плейлистом можна поділитися з іншими користувачами.
